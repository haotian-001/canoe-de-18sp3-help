Tips & Tricks

 

Valid for:  CANoe DE  •  CANoe:lite DE  •  CANoe4SW DE  •  CANoe4SW:lite DE

CAPL Object Files

Before you start a measurement you should verify that all CAPL files were compiled.

Saving Object References

Save object references in variables if you need the objects frequently.

In this context consider the following script code:

For Count = 1 To 5
  App.UI.Write.Output "Hello from VBScript: " & Count
Next

The second line is a (helpful and fully legal!) abbreviated notation. The code that is actually executed, however, appears (approximately) as follows:

For Count = 1 To 5
  Set ui = App.UI
  Set write = ui.Write
  write.Output "Hello from VBScript: " & Count
  Set write = Nothing
  Set ui = Nothing
Next

Please note that in each line there is a call to your CANoe DE product within the loop!
Consequently, a more efficient (and functionally fully equivalent) notation would be:

Set Write = App.UI.Write
For Count = 1 To 5
  Write.Output "Hello from VBScript: " & Count
Next
Set Write = Nothing

Again, you could call the first line at the start of your script and the last line at the end so that the task stated above would actually be reduced to the loop.
