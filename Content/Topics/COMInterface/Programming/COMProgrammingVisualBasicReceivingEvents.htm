Receiving Events (VB)

 

Valid for:  CANoe DE  •  CANoe:lite DE  •  CANoe4SW DE  •  CANoe4SW:lite DE

The following Visual Basic Code shows possibilities for responding to events within your CANoe DE product. The assumption is made that a form has been created with a TextBox and two switches (ConnectButton and ClearButton).

Note the declaration of global variables gApp and gMeasurement with the modifier WithEvents. This is necessary for receiving events. You can select the events that you want to receive directly in the lists of the code window (analogous to events of the controls).

Sample Code

Settings
Option Explicit

'Globals
Dim WithEvents gApp As <product COM name>.Application
Dim WithEvents gMeasurement As <product COM name>.Measurement

'Eventhandler
Private Sub ConnectButton_Click()
  Connect
End Sub

Private Sub ClearButton_Click()
  TextBox.Text = ""
End Sub

Private Sub Form_Load()
  TextBox.Text = ""
  Connect
End Sub

Private Sub Form_Unload(Cancel As Integer)Disconnect
End Sub

Private Sub gApp_OnOpen(ByVal Name As String)
'CANoe DE product has opened a configuration
  WriteLine ("CANoe DE product has opened a configuration.")
  If Name = "" Then
    WriteLine ("New configuration")
  Else
    WriteLine (Name)
  End If
End Sub

Private Sub gApp_OnQuit()
'CANoe DE product shuts down
  WriteLine ("CANoe DE product shuts down.")
  Disconnect
End Sub

Private Sub gMeasurement_OnExit()
'CANoe DE product is de-initializing a measurement
  WriteLine ("CANoe DE product is de-initializing a measurement.")
End Sub

Private Sub gMeasurement_OnInit()
'CANoe DE product is initializing a measurement
  WriteLine ("CANoe DE product is initializing a measurement.")
End Sub

Private Sub gMeasurement_OnStart()
'CANoe DE product is starting a measurement
  WriteLine ("CANoe DE product is starting a measurement.")
End Sub

Private Sub gMeasurement_OnStop()
'CANoe DE product is stopping a measurement
  WriteLine ("CANoe DE product is stopping a measurement.")
End Sub

'Subroutines
Private Sub Connect()
'Connect to CANoe DE product
Set gApp = New Application
Set gMeasurement = gApp.Measurement
ConnectButton.Enabled = False
WriteLine ("Connected to CANoe DE product.")
End Sub

Private Sub Disconnect()
'Disconnect from CANoe DE product
  Set gMeasurement = Nothing
  Set gApp = Nothing
  ConnectButton.Enabled = True
  WriteLine ("Disconnected from CANoe DE product.")
End Sub

Private Sub WriteLine(txt)
'Write a line to the text box
  If TextBox.Text = "" Then
    TextBox.Text = txt
  Else
    TextBox.Text = TextBox.Text & Chr(13) & Chr(10) & txt
  End If
End Sub
