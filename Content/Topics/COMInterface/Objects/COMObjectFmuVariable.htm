Description

The FmuVariable object provides access to the properties of a variable.

Property Value Element

Enabled

This property sets or returns whether the variable is enabled or not.

You should deactivate unused variables for performance reasons.

Syntax: object.Enabled[=value]

(see example below)

Type: BOOL

Indicates whether the variable is enabled during simulation; possible values:

The variable is active during simulation.

FALSE

The variable is ignored during simulation.
