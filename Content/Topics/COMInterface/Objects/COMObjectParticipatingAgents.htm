ParticipatingAgents

Application » Configuration » MultiCANoe » CoordinatorConfiguration » ParticipatingAgents

Valid for:  CANoe DE

Description

The ParticipatingAgents object represents the participating agents collection of the Multi CANoe cluster.

  • If the configuration contains invalid data, Multi CANoe will not be activated.

    As soon as the settings are valid, they are forwarded to the system and Multi CANoe can be executed properly.

  • If there is an error message, please resolve it to activate Multi CANoe.

Example

Example — General

This example shows how to create the ParticipatingAgents collection:

Set App           = CreateObject("CANoe.Application")
Set Configuration = App.Configuration
Set MultiCANoe    = Configuration.MultiCANoe

Set CoordinatorConfig   = MultiCANoe.CoordinatorConfiguration
Set ParticipatingAgents = CoordinatorConfig.ParticipatingAgents

' Now you can fill the ParticipatingAgents collection with agents
' Note that the collection already contains a default agent named "Agent1"

Example — Count property

This example shows how to return the number of available agents in the ParticipatingAgents collection:

Set App           = CreateObject("CANoe.Application")
Set Configuration = App.Configuration
Set MultiCANoe    = Configuration.MultiCANoe

Set CoordinatorConfig   = MultiCANoe.CoordinatorConfiguration
Set ParticipatingAgents = CoordinatorConfig.ParticipatingAgents

ParticipatingAgents.Add("CANoe_Agent1")
ParticipatingAgents.Add("CANoe_Agent2")

' Counts how many participating agents are in the collection. In this example it returns 3
WScript.Echo ParticipatingAgents.Count

' Remember that the collection already contains a default agent named "Agent1".

Example — Item property

This example shows how to return a specific object of the available agents in the cluster contained in the ParticipatingAgents collection:

Set App           = CreateObject("CANoe.Application")
Set Configuration = App.Configuration
Set MultiCANoe    = Configuration.MultiCANoe

Set CoordinatorConfig   = MultiCANoe.CoordinatorConfiguration
Set ParticipatingAgents = CoordinatorConfig.ParticipatingAgents

' Adds two agents to the collection with the instance name "CANoe_Agent1" and "CANoe_Agent2"
ParticipatingAgents.Add("CANoe_Agent1")
ParticipatingAgents.Add("CANoe_Agent2")

' Remember that the collection already contains a default agent named "Agent1".
' So, the first item of the collection is the ParticipatingAgent object named "Agent1".

' Returns the item (ParticipatingAgent object) "CANoe_Agent1" from the collection
WScript.Echo ParticipatingAgents.Item(2)

Example — Add method

This example shows how to add a new agent to the ParticipatingAgents collection:

Set App           = CreateObject("CANoe.Application")
Set Configuration = App.Configuration
Set MultiCANoe    = Configuration.MultiCANoe

Set CoordinatorConfig   = MultiCANoe.CoordinatorConfiguration
Set ParticipatingAgents = CoordinatorConfig.ParticipatingAgents

' Adds an agent to the collection with the instance name "CANoe_Agent"
ParticipatingAgents.Add("CANoe_Agent")

Example — Remove method

This example shows how to remove an agent by index from the ParticipatingAgents collection:

Set App           = CreateObject("CANoe.Application")
Set Configuration = App.Configuration
Set MultiCANoe    = Configuration.MultiCANoe

Set CoordinatorConfig   = MultiCANoe.CoordinatorConfiguration
Set ParticipatingAgents = CoordinatorConfig.ParticipatingAgents

ParticipatingAgents.Add("CANoe_Agent")

' This method deletes the agent you just created
ParticipatingAgents.Remove(2)

' Remember that the collection already contains a default agent named "Agent1".
' So, the first item of the collection is the ParticipatingAgent object named "Agent1".

Example — RemoveAll method

This example shows how to remove all agents from the ParticipatingAgents collection:

Set App           = CreateObject("CANoe.Application")
Set Configuration = App.Configuration
Set MultiCANoe    = Configuration.MultiCANoe

Set CoordinatorConfig   = MultiCANoe.CoordinatorConfiguration
Set ParticipatingAgents = CoordinatorConfig.ParticipatingAgents

ParticipatingAgents.Add("CANoe_Agent")

' This method deletes all the agents from the collection
ParticipatingAgents.RemoveAll()

Example — RemoveByName method

This example shows how to remove an agent by instanceName from the ParticipatingAgents collection:

Set App           = CreateObject("CANoe.Application")
Set Configuration = App.Configuration
Set MultiCANoe    = Configuration.MultiCANoe

Set CoordinatorConfig   = MultiCANoe.CoordinatorConfiguration
Set ParticipatingAgents = CoordinatorConfig.ParticipatingAgents
