CoordinatorConfiguration

Application » Configuration » MultiCANoe » CoordinatorConfiguration

Valid for:  CANoe DE

Description

The CoordinatorConfiguration object represents the coordinator part of the Multi CANoe configuration.

  • If the configuration contains invalid data, Multi CANoe will not be activated.

    As soon as the settings are valid, they are forwarded to the system and Multi CANoe can be executed properly.

  • If there is an error message, please resolve it to activate Multi CANoe.

Example

Example — General

This example shows how to create and configure the coordinator via MultiCANoe object:

Set App           = CreateObject("CANoe.Application")
Set Configuration = App.Configuration
Set MultiCANoe    = Configuration.MultiCANoe

' This sets the Multi CANoe configuration as a coordinator configuration and creates and CoordinatorConfig object
Set CoordinatorConfig = MultiCANoe.CoordinatorConfiguration

' With the CoordinatorConfiguration object you can now call the properties/methods of it

Example — ParticipatingAgents property

This example shows how to create the ParticipatingAgents collection:

Set App           = CreateObject("CANoe.Application")
Set Configuration = App.Configuration
Set MultiCANoe    = Configuration.MultiCANoe

Set CoordinatorConfig   = MultiCANoe.CoordinatorConfiguration
Set ParticipatingAgents = CoordinatorConfig.ParticipatingAgents

' Now you can fill the ParticipatingAgents collection with agents
' Note that the collection already contains a default agent named "Agent1"

Example — UseIPv6Protocol property

This example shows how to activate IPv6 and returns the value:

Set App           = CreateObject("CANoe.Application")
Set Configuration = App.Configuration
Set MultiCANoe    = Configuration.MultiCANoe

Set CoordinatorConfig = MultiCANoe.CoordinatorConfiguration

' Sets TRUE to activate IPv6
CoordinatorConfiguration.UseIPv6Protocol = TRUE

' Returns whether IPv6 is activated (=TRUE) or not (=FALSE)
WScript.Echo "Is UseIPv6Protocol: " & CoordinatorConfig.UseIPv6Protocol
