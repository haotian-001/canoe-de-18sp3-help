FmuKind

This property sets or returns the kind of the Fmu.

Each FMU can support multiple FMU kinds.

Attempts to set a value that is not supported by the FMU are silently ignored.

Therefore, the result of an assignment should be checked by a query.

Syntax: object.FmuKind[=value]

Example

Example — General

Disable all FMUs in the current FMI import configuration:

Set FmiConfiguration = App.Configuration.GetFmiConfiguration
For Each fmu in FmiConfiguration.FmuImport.FmuCollection
  fmu.Enabled = False
Next
App.Configuration.SetFmiConfiguration(FmiConfiguration)
