MultiCANoe

Application » Configuration » MultiCANoe

Valid for:  CANoe DE

Description

The MultiCANoe object represents the Multi CANoe settings within the configuration.

  • If the configuration contains invalid data, Multi CANoe will not be activated.

    As soon as the settings are valid, they are forwarded to the system and Multi CANoe can be executed properly.

  • If there is an error message, please resolve it to activate Multi CANoe.

Example

Example — General

This example shows how to create the MultiCANoe object.

Set App           = CreateObject("CANoe.Application")
Set Configuration = App.Configuration
Set MultiCANoe    = Configuration.MultiCANoe

Example — AgentConfiguration property

This example shows how to create and configure an agent via MultiCANoe object:

Set App           = CreateObject("CANoe.Application")
Set Configuration = App.Configuration
Set MultiCANoe    = Configuration.MultiCANoe

' This sets the Multi CANoe configuration as an agent and creates an AgentConfiguration object
Set AgentConfig = MultiCANoe.AgentConfiguration

Example — ClusterState property

This example displays the current state of the Multi CANoe cluster:

Set App           = CreateObject("CANoe.Application")
Set Configuration = App.Configuration
Set MultiCANoe    = Configuration.MultiCANoe

' Enables Multi CANoe
MultiCANoe.Enabled = TRUE

' Returns current state of the Multi CANoe cluster
WScript.Echo "Cluster state: " & MultiCANoe.ClusterState

Example — CoordinatorConfiguration property

This example shows how to create and configure the coordinator via MultiCANoe object:

Set App           = CreateObject("CANoe.Application")
Set Configuration = App.Configuration
Set MultiCANoe    = Configuration.MultiCANoe

' This sets the Multi CANoe configuration as a coordinator configuration and creates and CoordinatorConfig object
Set CoordinatorConfig = MultiCANoe.CoordinatorConfiguration

Example — Enabled property

This example shows how to enable and return the state of the Multi CANoe setting:

Set App           = CreateObject("CANoe.Application")
Set Configuration = App.Configuration
Set MultiCANoe    = Configuration.MultiCANoe

' Enables Multi CANoe configuration
MultiCANoe.Enabled = TRUE

' Returns state of Multi CANoe configuration
WScript.Echo "State: " & MultiCANoe.Enabled

Example — FDXPort property

This example shows how to set and return the FDX port:

Set App           = CreateObject("CANoe.Application")
Set Configuration = App.Configuration
Set MultiCANoe    = Configuration.MultiCANoe

' Sets FDXPort
MultiCANoe.FDXPort = 2809

' Returns FDXPort
WScript.Echo "Is FDXPort: " & MultiCANoe.FDXPort

Example — StartTimeout property

This example shows how to set and return the timeout within the measurement must start:

Set App           = CreateObject("CANoe.Application")
Set Configuration = App.Configuration
Set MultiCANoe    = Configuration.MultiCANoe

' Sets StartTimeout
MultiCANoe.StartTimeout = 6

' Returns StartTimeout
WScript.Echo "StartTimeout: " & MultiCANoe.StartTimeout

Example — OnClusterStateChanged event

This example shows a procedure which is called when the current state of the Multi CANoe cluster changes:

Set App           = CreateObject("CANoe.Application")
Set Configuration = App.Configuration
Set MultiCANoe    = Configuration.MultiCANoe

' Binds the events of the MultiCANoe object to event handlers (sinks) in the currently executing script
WScript.ConnectObject MultiCANoe, "MultiCANoe_"

' Enables Multi CANoe
MultiCANoe.Enabled = TRUE

' Procedure which is called when the cluster state of Multi CANoe changes
Sub MultiCANoe_OnClusterStateChanged(state)
  WScript.Echo state
End Sub

' Keep in mind! Keep the script execution running
