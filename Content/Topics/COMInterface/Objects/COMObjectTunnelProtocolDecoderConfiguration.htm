TunnelProtocolDecoderConfiguration

 

Valid for:  CANoe DE  •  CANoe4SW DE

Description

The TunnelProtocolDecoderConfiguration object represents the configuration of the tunnel protocol decoder.

ExportConfiguration

This method exports the current tunnel protocol decoder configuration to a JSON file.

This file can then be used in different configurations (e.g. CANalyzer, CANoe DE, CANape) by importing it again.

Syntax: object.ExportConfiguration(path)

(see example below)

Example

Example — General

Copy

The following example written in PowerShell demonstrates the usage of various methods of the tunnel protocol decoder configuration instance:
(see note on <product COM name>)

# Get the tunnel protocol decoder configuration instance
$canoe = New-Object -ComObject <product COM name>.Application
$tunnel = $<product COM name>.Configuration.TunnelProtocolDecoderConfiguration

# Export the current tunnel protocol decoder configuration
$tunnel.ExportConfiguration("D:\TunnelProtocolDecoderConfiguration.json")

# Import a configuration
$tunnel.ImportConfiguration("D:\TunnelProtocolDecoderConfiguration.json", $true)
