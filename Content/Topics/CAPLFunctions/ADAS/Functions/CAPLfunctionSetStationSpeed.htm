SetStationSpeed

 

Valid for:  CANoe DE

Function Syntax

Int64 SetStationSpeed(char* stationName, double speed, double acceleration);

Description

With this function the speed of a station can be changed. The parameter speed defines the speed that has to be reached with a given acceleration (2nd parameter).

Note

This function does not work for dynamically added stations because their speed is defined by the frequency their positions are provided.

Parameters

stationName

Name of the station. The name must match the name of the node in the database.

speed

The new speed of the station in km/h.

acceleration

The used acceleration in m/s2 to reach the new speed (in case of deceleration, this will be calculated automatically).

Return Values

0

OK

-1

Error

Example

on start
{
  SetStationSpeed("Station1", 200.0, 15.0);
}
