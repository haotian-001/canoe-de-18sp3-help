Command Line Parameters

 

 

Following command line parameters are supported:

Parameter

Valid for:  

Note

a

CANoe DE

CANoe:lite DE

CANoe4SW DE

CANoe4SW:lite DE

The measurement is started automatically.

f

CANoe DE

CANoe:lite DE

CANoe4SW DE

CANoe4SW:lite DE

The defined configuration is loaded.

Example

  • CANoe64.EXE /f "D:\CANSystemDemo\CANSystemDemo.CFG"

  • CANoe4SW64.exe /f "D:\SomeDirectory\MyConfiguration.cfg"

m

CANoe DE

CANoe:lite DE

Your CANoe DE product starts with the specified application name.

Example

CANoe64.exe /m MasterCANoe

Note

The application name is used for channel assignment in the Vector Hardware Config dialog.

Especially if multiple instances are operated on one computer, these instances must be started with different application names.

One use case for this is Multi CANoe.

n

CANoe DE

CANoe:lite DE

A prefix is added to all specified Ethernet networks in a configuration when it is opened.

The prefix consists of the application name (see under parameter m), a separating "_" and the original network name.

If no parameter n is specified, all Ethernet networks in the configuration receive the prefix.

Note

The prefix is active for every configuration that is opened with the CANoe instance that has received the arguments. If the channel assignment is changed manually in the current configuration, the prefixes for the changed networks are no longer used. A prefix for an NDIS network is not functionally supported.

One usage scenario for this argument is serial tests, where the same configuration can be used for all connected devices, only the hardware networks have slightly different names.

If a channel assignment error occurs after changing the device assignment, you must reload the configuration.

Example

  • CANoe64.exe /m MasterCANoe /n "Ethernet1"

  • CANoe64.exe /m MasterCANoe /n

r

CANoe DE

CANoe:lite DE

CANoe4SW DE

CANoe4SW:lite DE

The last configuration is automatically loaded at startup.

c

CANoe DE

CANoe:lite DE

The application starts in distributed mode using a designated remote address.

Example

CANoe64.exe /c 192.168.0.2:2809

Note on Product Variants (Available for CANoe DE, CANoe:lite DE)

To limit the application to a certain variant, the following arguments are available to you:

  • CANoe64 /limitto:<variant> with <variant>=pex or <variant>=run

Example

CANoe64.exe /limitto:pex
