Working with an Associated Database

 

Valid for:  CANoe DE  â¢  CANoe:lite DE

If there is a database associated to the configuration, messages from the database can be added to the transmit list. After clicking the [New] button, the Symbol Selection dialog is opened, with which messages can be selected symbolically. These are then added to the transmit list. Afterwards, signals of the messages can be configured individually in each message's signal list. As an alternative to the [New] button, double clicking the first blank line in the "Message Name" column with the left mouse button will also open the Symbol Selection dialog.

In the "Chip" column of message parameters you can configure the channel/chip over which the message is to be transmitted. This setting is taken from the database allocation to a channel. If the database is not allocated to a particular channel, "Chip 1" is configured as the default; it is then possible to change this setting.

Options can be preconfigured for messages and signals in the database. It is possible to define the cycle time for messages and signal values as database attributes and assign default values.

The attributes for messages are:

  • CycleTime: Integer [ms].
    Explicit cycle time for the Interactive Generator.

The attributes for physical signal values are:

  • SignalDefault: Float [phys. signal value].
    Signal value for the Interactive Generator.

If both values are set, the attribute for the Interactive Generator has a higher priority and is used. If neither of the attributes is set, the previous default values are used.
