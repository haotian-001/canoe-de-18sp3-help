CANopen Interactive Generator (CANopen IG)

The CANopen Interactive Generator enables the creation of message sequences that can be sent once or cyclically. Each type of CANopen specific message can be combined into a sequence. The messages are configured via dialogs.

Insert the CANopen Interactive Generator

The CANopen Interactive Generator can be inserted into the Simulation Setup. Right-click on the line that symbolizes the bus and select the Insert CANopen Interactive Generator command. The CANopen Interactive Generator can be opened in the Simulation Setup with a double click on the symbol.

The messages are sent on the CAN channel that is assigned to the network in the System View.

Message Sequences

You can add CANopen messages to a sequence using the toolbar or the context menu. You can also reopen the configuration dialog displayed by double-clicking on the corresponding line.

You can activate or deactivate parts of a sequence via the Enabled column.

The button in the Send column sends the selected message while the measurement is running.
