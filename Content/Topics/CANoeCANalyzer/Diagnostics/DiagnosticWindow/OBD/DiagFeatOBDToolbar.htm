Toolbar (OBD - Diagnostic Feature)

Ribbon|Diagnostics & XCP Tab|Diagnostic Window|OBD

Valid for:  CANoe DE

Symbol

Function

Network Scan

Performs a network analysis. A diagnostic query (via OBD-II or OBDonUDS) searches for OBD-capable ECUs in the network and the OBD functions they support.

Update OBD-Data

Reads the data for all selected PIDs, OBDMIDs and Info Types.

Cyclically Update of Live Data

Reads the data for all selected PIDs with the set cycle time.

Oxygen Sensor Legend on/off

Shows or hides the oxygen sensor legend.

Create OBD Report

Creates an XML export of the current data from the Diagnostic Feature OBD.

Open Options Dialog

Opens the dialog for configuring the Diagnostic Feature OBD.
