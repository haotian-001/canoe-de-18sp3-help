Toolbar (Fault Memory - Diagnostic Feature)

Ribbon|Diagnostics & XCP Tab|Diagnostic Window|Fault Memory|Toolbar (Fault Memory - Diagnostic Feature)

Valid for:  CANoe DE

Symbol

Function

Read Fault Memories of all configured ECUs

Reads in the fault memory entries

Read cyclic Fault Memories of all configured ECUs

If this option is activated, the fault memory is read out at the intervals set next to it.

Read mode "Identified DTCs by status"

The diagnostic service is used for reading, with which only fault memory entries in the primary fault memory that match the set status are read out.

Read mode "Identified DTCs by severity"

The diagnostic service is used for reading, with which only fault memory entries matching the set severity level are read out.

Read mode "Identified DTCs (User Memory) by status"

The diagnostic service is used for reading, with which only fault memory entries in the user-defined fault memory that match the set status are read out.

Read mode "Supported DTCs"

The diagnostic service is used for reading, with which the status of the supported fault memory entries in the primary fault memory is read out.

—

Status

This is where the status bits are set that are used when querying the fault memory for some read operations.

—

Severity

Here you can set the severity level that is used when querying the fault memory for some read operations.

Clear diagnostic trouble code

The selected DTC is deleted.

Clear all diagnostic trouble codes

All DTCs for the diagnostic descriptions/ECUs configured in this diagnostic windows are deleted.

Search in fault memory entries

Opens or closes the search, with which the fault memory entries can be searched using a full-text search. Multiple search terms can be disconnected by a space. Search terms that contain spaces must be enclosed in quotation marks.

Search in snapshot records, extended data records or additional information

Opens or closes the search, with which the snapshot records, extended data records or additional information can be searched using a full-text search. Multiple search terms can be disconnected by a space. Search terms that contain spaces must be enclosed in quotation marks.

Open Fault Memory configuration

Opens the configuration dialog of the fault memory.

Delete Diagnostic Trace content

Deletes the diagnostic Trace area.

Show only failed

Displays only failed messages.

Export Diagnostic Trace content

Exports the content of the diagnostic trace, taking into account the filter settings.
