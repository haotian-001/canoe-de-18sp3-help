Synchronize Windows

Context Menu of the Test Execution dialog: Synchronize Windows ((available for CANoe DE))

Valid for:  CANoe DE  •  CANoe:lite DE  •  CANoe4SW DE  •  CANoe4SW:lite DE

You would like to synchronously analyze data in several analysis windows when a measurement is stopped?

The function Synchronize Windows is only available for windows that can display a time characteristic:

  • Trace Window

Note

In fixed position mode, the Trace Window is not synchronized with other windows, because no data history is available in this mode.
  • Test Trace Window
  • Graphics Window
  • Map Window
  • State Tracker
  • Test Execution dialog (test modules) (available for CANoe DE)
  • Test Execution dialog (test units)
  • Video Window (available for CANoe DE)

Except for the Video Window you can activate or deactivate the Synchronize windows function via the context menu of a window.

Alternatively, you can find the icon in the Home ribbon tab, which activates or deactivates the function globally for all windows.

Example of Use

The Synchronize Windows function can be used very effectively in connection with the Trace Window, especially when you activate Continuous Mode there: . If you then click on a line in the Trace Window, blue measurement cursors are displayed at the synchronous time in the other synchronized analysis windows. You can then move the cursor in the Trace Window using the arrow keys on your keyboard, and the measurement cursor in the other windows will be moved with too.

Conversely, when the Synchronize Windows function is activated, you can also move the blue measurement cursors in the synchronized windows to the left or right (e.g. Graphics Window) and the cursor in the Trace Window will also move.
