Detail View

 

Valid for:  CANoe DE

The Detail View shows more information about the currently selected object.

If a property refers to another object, this can be selected directly in all views via the context menu command Select (depending on the network selection, grouping and filter settings). An object filter for the Communication View can also be created using the context menu command Create Object Filter.

Note

The fully qualified name is displayed for:

  • Networks
  • Endpoints (in tooltip)
  • Connections (in tooltip)

Individual lines or the entire view can be copied to the clipboard as text via the context menu. In addition, individual elements or the entire view can be expanded or collapsed.

If necessary, automatically switches to another page (Communication View, Validation Overview, ...) and centers the view on the object.

Endpoint status:

  • Up: the existence of the endpoint has been actively confirmed (e.g., by an event sent by the endpoint).
  • Down: the existence of the endpoint has been explicitly denied (e.g. by a management protocol)
  • Pending: the existence of the endpoint is unknown (e.g. because it has only appeared as a receiver so far)
