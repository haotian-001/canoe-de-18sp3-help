Validation Overview

 

Valid for:  CANoe DE

The Validation Overview lists all discrepancies observed in the data traffic as validation results.

With the context menu command Create Object Filter you can create an object filter for the Communication View, which reduces its content to the essential elements.

Column

Description

Severity

Severity level of the validation result:

  • Information
  • Warning
  • Error

Time [s]

Last time at which the validation result was detected.

Status

Status of the validation result:

  • Unresolved: the observed problem has not been resolved
  • Resolved: the observed problem has been resolved

Note

Not every type of validation results can be resolved.

Name

Short name of the validation result.

Description

Detailed description of the validation result.

Column Filter

You can configure a column filter for each column to filter the displayed validation results based on their properties.

If you move the mouse button over a table heading, a filter symbol appears in the top right-hand corner. You can configure or reset the filter by clicking on the filter icon. The configured column filters only affect the Validation Overview and not the other available views.

You can reset all column filters via the context menu.
