SOME/IP Overview

 

Valid for:  CANoe DE

The SOME/IP Overview offers an alternative way to start analyzing SOME/IP communication. It lists all observed services, service instances, consumers, providers, subscriptions and eventgroup subscriptions in a hierarchical table.

If an object is selected, further information is displayed in the Detail View, such as the network nodes, networks, endpoints and connections involved. With the context menu command Create Object Filter you can create an object filter for the Communication View, which reduces its content to the essential elements.

SOME/IP Object Types

The Protocol Monitor distinguishes between the following SOME/IP object types.

Services form the first level. The observed service instances are listed under each service. The observed consumers, providers and subscriptions are listed under each service instance. The eventgroup subscriptions are listed under each subscription.
