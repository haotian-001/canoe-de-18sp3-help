Tester and Reset Parameters

Ribbon: Test Tab|Conformance Test Configuration

Valid for:  CANoe DE

The reset parameters configure the restart of the responder, that is carried out before each test case. The reset can be performed via the VH1100 family, a VT7001 module, or manually. The reset can also be replaced by a software reset (bus idle) or omitted.

The tester parameters configure settings to automate the test procedure. To use all options, a VH1150 or newer device is needed.

When using VT7001 or the VH1100 family, the current consumption of the responder can be used to automatically determine whether sleep mode has been entered.

For the J2602 conformance tests, it is possible to use automatic adjustment of the dominant/recessive bus level.

Parameter

Value Range

Description

Reset_Type

[Manual, Software, None, VH11xx, VT7001]

Defines how the responder is restarted:

  • Manual: Responder is restarted manually.

  • Software: Instead of a restart, bus idle is kept for a definable time.

  • None: No restart.

  • VH11xx: Interruption of the power supply via VH11xx.

  • VT7001: Interruption of the power supply via VT7001.

Reset_VT7001_Name

Modul name of the VT7001

Only for reset via VT7001.

Name of the VT7001 module to which the responder power supply is connected.

Reset_VT7001_OutChannel

Channel for the power supply of the responder on the VT7001

Only for reset via VT7001.

Name of the VT7001 channel to which the responder is connected.

Important: The measured AvgCurrent value should be activated on the output channel used, see VT System configuration.

Reset_IUTPowerOffTime_ms

[0…]

Duration of the power interruption during reset in milliseconds.

Reset_WaitForBusIdleTime_ms

[0…]

Duration of bus idle time for software reset in milliseconds.

Reset_IUTInitTime_ms

[4…120]

Duration of the minimum waiting time until bus traffic after switching on the power supply in milliseconds.

Tester_SupplyVoltage_V

[0,0…28,0]

Supply voltage of the control unit in volts.

Tester_AutomaticSleepAwakeDetection

[false, true]

Only for reset via VT7001 or VH11xx.

true: Switching the responder to sleep mode is automatically detected based on the current consumption.

false: Manual detection of the sleep mode.
Tester_IUTSleepCurrentConsumptionMax_mA

[0,0…2000.0]

Maximum current consumption of the responder in sleep mode in milliamperes.

Tester_IUTAwakeCurrentConsumptionMin_mA

[0,0…2000.0]

Minimum current consumption of the responder in awake mode in milliamperes.

Tester_TimeToLowPowerState_ms

[150…1000000]

Time limit for reducing the current consumption after switching to sleep mode in milliseconds.

Tester_UseAutomaticBusLevelAdjustment

(only SAE J2602)

[false, true]

true: Requires VH1150. Automatically adjusts the dominant/recessive bus level for the J2602 test cases that required it.

false: Manual adjustment.
