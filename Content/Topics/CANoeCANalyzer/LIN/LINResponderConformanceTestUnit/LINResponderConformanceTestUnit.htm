LIN Responder Conformance Test Unit

Ribbon: Test Tab|Conformance Test Configuration

Valid for:  CANoe DE

General

The LIN Responder Conformance Test Unit can be used to perform conformance tests within a CANoe DE configuration. You can add and open test configurations for conformance test units via the menu path Test Tab|Conformance Test Configuration. You can manage the conformance test configurations in the Test Setup for test units. If a new test configuration is created, a new responder conformance test unit is automatically added for the selected responder.

The test cases are implemented in CAPL libraries, which are automatically assigned to the test module. The test cases can be controlled via the test module's execution window. It is currently possible to perform slave conformance tests for the LIN protocol versions 2.x, ISO17987, SAE J2602:2012, and SAE J2602:2021. The Vector hardware VN1611 is currently not supported.

Notes for LIN 2.1

With the VN modules (VN8900, VN1600) all tests (including the sampling point test 3.17) can be performed strictly according to the LIN2.1 conformance test specification.

Notes on Test Coverage

  • The responder conformance test unit supports tests for layer 2 and 3.
  • Physical tests and EMC tests are not supported.

Notes

  • To perform all tests, VN modules (VN8900, VN1600) are required and recommended. The hardware has extended stress functionality and higher accuracy.

  • Many tests can be automated using VH1100 test hardware or the VT7001 module of the VT System family. Both can automatically reset the IUT and detect sleep/awake mode by measuring the current consumption of the IUT.

Configuration of the Conformance Test Units

When a conformance test unit is created, the appropriate version of the conformance test and the test cases that correspond to the parameters of the selected responder are automatically selected.

Open a test configuration via the menu path Test Tab|Conformance Test Configuration to change the test cases to be executed. The execution tree is displayed on the left-hand side of the window. You can activate or deactivate the execution of the individual test cases and test groups using the check boxes.

In the lower part of the window, you can use the Variant properties profile to change the parameters of the responder and the tests. A description of the parameters can be found on the following help pages:

  • Responder Parameters

  • Tester and Reset Parameters

  • Test Report Parameters

  • ISO17987/LIN 2.1 Parameters

  • SAE J2602 Parameters

  • LIN 2.0 Parameters
