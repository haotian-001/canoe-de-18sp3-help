SAE J2602 Parameters

Ribbon: Test Tab|Conformance Test Configuration

Valid for:  CANoe DE

Parameters for protocol versions SAE J2602-2012 and SAE J2602-2021.

Parameter

Value Range

Description

IUT_Autobauding_Supported

[false, true]

true: Responder supports autobauding.

false: No autobauding test cases.

IUT_BaudrateDetection_Supported

(only for J2602:2021)

[false, true]

true: Responder supports baudrate detection.

false: No baudrate detection test cases.

IUT_HardwareSelectableDNN

[false, true]

true: DNN of the responder can be defined via hardware (pins, switches, etc.).

false: No test cases for selecting the DNN via hardware.

IUT_SoftwareSelectableDNN

[false, true]

true: DNN of the responder can be defined via software (programmable).

false: No test cases for selecting the DNN via software.

IUT_AssignNAD_Supported

[false, true]

true: Responder supports the AssignNAD command.

false: No support.

IUT_AssignFrameID_Supported

(only for J2602:2012)

[false, true]

true: Responder supports the AssignFrameID command.

false: No support.

IUT_AssignFrameIDRange_Supported

(only for J2602:2021)

[false, true]

true: Responder supports the AssignFrameIDRange command.

false: No support.

IUT_SleepTimeout_s

(only for J2602:2012)

[4…120]

The duration of bus idle in seconds at which the responder must switch to sleep mode at the latest.

Tester_ManualVerificationViaScope

[false, true]

true: Activates the manual verification of some test cases with Option Scope.

false: Manual execution of test cases without the Scope option.

Report_AddScopeSnapshotOnManualVerification

[false, true]

true: For manual verification with the Scope option, a screenshot of the Scope window is added to the report even if the test case is successful.

false: No screenshot if the test case is successful.

Tester_UseExternalBusload

[false, true]

true: Deactivates the internal master resistor of the LIN interface.

false: Internal resistor stays active.

Usage of Option Scope for Verification (only J2602:2012 and J2602:2021)

Test cases of the J2602 Conformance Tests requiring manual verification can be performed using option Scope. The scope will be triggered at the correct time if manual verification is needed. This is possible for the following test cases:
