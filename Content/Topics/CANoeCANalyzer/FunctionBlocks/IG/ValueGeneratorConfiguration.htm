IG: Signal Configurator Configuration

 

Valid for:  CANoe DE  •  CANoe4SW DE

You open this dialog via the [Define...] button in the signal list.

You can define the signal chart and test the response of an ECU with the help of the Signal Generator.

Depending on the generator type, you can determine whether the signal chart should be sent

  • Only once or repeatedly

  • Time bound or not time bound

A list of the calculated and output values as well as a graphic representation makes it easier for you to evaluate the defined signal chart.

Because not all signal chart values are output in the case of a time-dependent transmission, but instead only the calculated value at the cyclical transmission time, the time [ms] is displayed in the value list in the case of an activated time-dependent transmission.

When defining the signal chart, you are free to use the default generators or enter/import your own values.

With [Copy All] you can copy the values within your CANoe DE product and e.g. reuse and modify them in the User-defined generator type in the Values section by inserting them with the context menu command Paste.

Possible Generator Types

Sine

  • This generator type produces a sine function.

Value Range

  • This generator type generates a signal curve whose value range is defined in the CANdb++ database.

Ramps and Pulses

  • This generator type produces ramps and pulses.

Random

  • This generator type sends random numbers between two values.

    You can use this random curve e.g. for system stress tests or for testing reactions to undefined events.

Toggle Switch

  • With this generator type, the generator switches between two values and is therefore suitable for testing bit signals, for example.

User-defined

  • With this generator type, you can create any signal curves and change recorded signal curves in a targeted manner.

Variable

  • This generator type receives its value(s) from the selected system variable.

    The values of the variables can be entered via a panel, for example.
