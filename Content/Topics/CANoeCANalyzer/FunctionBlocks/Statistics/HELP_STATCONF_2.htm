Statistics Report

Context Menu of the Frame Histogram in Measurement Setup: Display Statistics Report

Valid for:  CANoe DE  â¢  CANoe:lite DE  â¢  CANoe4SW DE  â¢  CANoe4SW:lite DE

If Statistics Report is activated, after a measurement has been run a report of statistical data can be requested via the context menu of the Frame Histogram in the Measurement Setup. This report is output in the Write window.

Output in the first line are the last valid acquisition range and the time at the measurement start.

Output in the first two columns are the identifiers and message attributes of the received messages.

The other columns have the following meanings:

N

Number of received messages with this ID and attribute

Aver

Mean value of the transmit interval in ms

StdDev

Associated standard deviation in ms

MIN

Shortest transmit interval between two messages of this type in ms

MAX

Longest transmit interval between two messages of this type in ms

Shown in the last column is the CAN channel over which the specific event occurred.

The Delay line represents a statistic for the transmit interval (Time between Tx and TXRQ).
