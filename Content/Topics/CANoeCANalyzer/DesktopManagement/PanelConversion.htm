Conversion of Panel Editor Panels

 

Valid for:  CANoe DE  •  CANoe:lite DE

As of version 9.0, the Panel Editor and panels created with the Panel Editor are no longer supported.

Old panels are converted automatically by opening them in the Panel Designer.

NM controls will be converted automatically since version 13.0.

Conversion of Panels: CNP to XVP

If you have decided to convert, note the following:

  • The converted panels are located in the same directory as the Panel Editor panels.

  • The Panel Editor panels are removed from the configuration but the files are retained.

  • The converted panels are added to the configuration.

  • Panel Designer panels that reference Panel Editor panels (Panel Control Button) are overwritten and reference the corresponding converted panels.

  • Check the converted panels since not all panel settings can be completely converted.

  • After the conversion operation, store your configuration.

You will find information on the conversion operation in the Write Window.

During the conversion a report (<ConfigurationName>_PanelConversionLog.txt) is generated containing the following information:

  • Date and time of the conversion

  • Information on each panel that was converted (regardless of whether or not errors occurred)

  • Listing of the Panel Designer panels that were added to the configuration (including referenced panels)

  • Listing of the Panel Editor panels that were removed from the configuration (including referenced panels)

The report is stored in the same directory as the configuration.

Errors during Conversion

If an error occurred during conversion, this is indicated in a dialog. The additionally generated report (<PanelName>_Conversion_Report.txt) can be opened with [Open Log File]. The report will be saved in parallel to the panel file.
