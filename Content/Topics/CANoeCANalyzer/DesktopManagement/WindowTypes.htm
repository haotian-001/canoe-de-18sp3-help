Window Types

 

Valid for:  CANoe DE  •  CANoe:lite DE  •  CANoe4SW DE  •  CANoe4SW:lite DE

Various types are available for the windows, each of which defines a particular window behavior. You can change the window type via the context menu (opened by right-clicking the title bar).

MDI Window

  • standard form of newly displayed windows

  • can only be displayed within the main window

Standard Window

  • can be positioned anywhere on the Windows desktop

  • is independent from the main window and has its own button on the task bar

  • can be attached to the main window

  • can be maximized while the main window is minimized

Docking Window

  • can be docked to predefined positions in the main window using a drag and drop operation

  • can be grouped with another docked window

  • can be set to Auto-Hide mode

Position Docking Windows

Click the title bar and drag the window to the desired position. The possible positions for an automatic layout are shown in a preview when dragging. Drag the mouse directly to a preview icon to dock the window there, e.g., at the left edge with this icon :

Group Docking Windows

You can group several docked windows by dragging the windows directly one above the other. In this case, only one of the grouped windows is displayed at a time. You can call the others using the automatically generated pages. See below: Data and Trace pages:

Auto-Hide Docking Windows

Docked windows can be switched to Auto-Hide mode using the Auto-Hide button (see above). In Auto-Hide mode, as soon as another window is clicked, the windows are reduced to pages at the edge of the main window. You can use this mode to save space if you require windows only occasionally.

'Float' Docking Windows

If a window with the Docking Window setting is not docked anywhere, it is a so-called Floating Window.

A Floating Window:

  • can be completely removed from the main window

  • is always in front of the main window

  • is minimized together with the main window
