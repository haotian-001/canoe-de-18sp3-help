Display

 

Valid for:  CANoe DE

Display Modes

The different modes are displayed at the top left of the scene view when you hover over them with the mouse.

The Bird's Eye mode shows objects in two dimensions according to their length and width.

The Default mode contains a three-dimensional representation of the detected objects from the view slightly behind and above the host vehicle.

The Driver mode contains a three-dimensional representation of the detected objects from the driver's perspective.

The Rear mode contains a three-dimensional representation of the detected objects located behind the host vehicle.

Further Modes

  • Top
    Similar to Bird's Eye mode.
  • Left
    Shows the host vehicle and the detected objects on a flat plane as seen from the left.
  • Right
    Shows the host vehicle and the detected objects on a flat plane as seen from the right.
  • Back
    Shows the host vehicle and the detected objects on a flat plane as seen from behind.
  • Front
    Shows the host vehicle and the detected objects on a flat plane as seen from the front.

Display Settings

You can set the following options via the display settings in the toolbar of the Scene Window:

  • Show Inactive Objects
  • Show Ground Truth Data
  • Show Velocity Indicator X-Axis
  • Show Velocity Indicator Y-Axis

If the Show Inactive Objects option is enabled, the Detected Object view will also show objects that were detected in the past and are no longer detected.

If the option Show Ground Truth Data is activated, ground truth data will be displayed in addition to the detected objects. Ground truth data is modeled as environment objects within the ADAS feature. An overview of the individual object types in the ADAS data model can be found on the page Detected Sensors and Objects.

If the option Show Velocity Indicator X-Axis is activated, an arrow is drawn for each detected object, which shows the relative speed in the X direction. The length of the arrow reflects the relative speed in m/s.

Examples

  • Relative speed to host vehicle is -5 m/s: The arrow is 5 m long and points towards the host vehicle.
  • Relative speed to host vehicle is 8 m/s: The arrow is 8m long and points away from the host vehicle.

If the option Show Velocity Indicator Y-Axis is activated, an arrow is drawn for each detected object, which shows the relative speed in the Y direction. The length of the arrow reflects the relative speed in m/s.

Examples

  • Relative speed to host vehicle is -5 m/s: The arrow is 5 m long and points to the right from the host vehicle.
  • Relative speed to host vehicle is 8 m/s: The arrow is 8m long and points to the left from the host vehicle.

Color Settings

The Color Settings for Objects toolbar of the Scene Window opens a configuration dialog.

In this, a color mode for Detected Objects, Ground Truth Objects or the speed indicators can be selected. Either Sensor Based or Type Based can be selected.

  • With the Sensor Based option, the Detected and Ground Truth Objects have the same color as the sensors that detect them.
  • With the Type Based option, a color can be selected for each object type. The setting of the colors is also done in this dialog.
