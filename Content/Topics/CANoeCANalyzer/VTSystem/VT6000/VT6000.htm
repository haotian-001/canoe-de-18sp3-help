VT6000/VT6020/VT6060 – Realtime Module

 

Valid for:  CANoe DE  •  CANoe:lite DE

General

The realtime module has an embedded computer that runs the realtime specific parts of tests and simulations in the VT System rack. The user sees the VT6000 as a pre-configured RT server. The user computer (distributed mode) is connected to the VT6000 via Ethernet.

The realtime module comes with a range of processor boards and components, which affects the number of network interfaces that can be connected to it directly. These different versions are distinguished from one another by means of the last two places in the VT number (e.g. VT6020, VT6060). They do not differ in their usage, however. The term VT6000 encompasses all versions of the realtime module.

The variant VT6060 supports Extended Real Time (ERT). Please also see page VT System and Extended Real Time.

Switching Different Signal Paths

Technical Data

The realtime module serves as a computational module that is integrated into the VT System rack. Hardware signals are not routed via the VT6000.
