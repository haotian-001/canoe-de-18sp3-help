System Variables

 

Valid for:  CANoe DE  •  CANoe:lite DE

The following values are available via system variables for the channels of a VT2812 module.

The namespace is the name of the channel in the VT System configuration:

Value/Setting

System Variable

R/W

Value Semantic

Currently output voltage value.

CurOutputVoltage

R

Double, in Volt

(-10.0…28.0)

Stimulation output status

StimulationRunning

R

Integer

(0 = off, 1 = running)

Relay to Gnd

(The closed relay connects line b of the channel with the externally connected common ground.
This lets you use the ground as a reference potential for the output.)

RelayGnd

W

Integer

(0 = open, 1 = closed)

Relay to DGND

(The closed relay connects line b of the channel with the internal reference potential DGND.
This lets you use DGND as the reference potential for the output.)

RelayDGnd

W

Integer

(0 = open, 1 = closed)

The following settings can be written or read via system variables for a VT2816 module.

The setting is valid for the entire module. The namespace is the name of the channel in the VT System configuration:

Value/Setting

System Variable

R/W

Value Semantic

This variable indicates if this channel/module operates in offline or online mode.

IsOffline

R

Boolean

It is not possible to read back output values.

When the various system variables are read, only the most recent value is returned. However, this value does not always represent the current state of the hardware.
