VT2812 – General-Purpose Analog Output Module

 

Valid for:  CANoe DE  •  CANoe:lite DE

General

The VT2812 General-Purpose Analog Output Module contains 12 identically structured channels for the output of analog voltages.

The available output range extends from -10 V to +28 V.

Switching Different Signal Paths

Analog voltages can be output on all channels of the VT2812.

The VT2812 does not have a busbar for external voltages.

However, you can switch line b of the output channels to DGND or a common potential, which can be applied externally.

Notice!

Correct grounding is absolutely essential when operating VT System modules!

In particular, you need to connect AGND with DGND.

You also need to make sure to connect the ECU ground to the VT2816 when the output channels are not operated in differential mode.

If these connections are missing, the VT System module may be damaged!

For more information, see VT System Ground Connection.
