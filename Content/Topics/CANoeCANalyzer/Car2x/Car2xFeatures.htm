Analysis

Simulation/Stimulation

The windows and blocks in the measurement setup offer extended functions for Car2x:

Trace Window

  • Evaluation of WLAN packets
  • Column set Car2x
  • Automatic protocol checks

Map Window

  • Display of location coordinates in the Car2x network

Graphic Window, Data Window and State Tracker

  • Signals defined in the database can be displayed

Logging

  • Log and replay network traffic

CAPL

  • Specific Car2x CAPL functions for different areas
  • Callback functions for responding to messages
  • Packet API for accessing messages
  • Security API for runtime analysis

Security

  • Analyze certificates at runtime
  • CAPL Security API for runtime analysis

Each node in the Simulation Setup can be configured individually.
