Migration of Certificates to the Vector Security Manager

 

Valid for:  CANoe DE

The Car2x Certificate Manager function has been integrated into the Vector Security Manager with release 18. This procedure describes how you can migrate your existing configuration.

Cross Reference

The creation of a new configuration with Car2x Security is described under Creating a Security Profile in the Vector Security Manager.

To migrate a configuration to the Vector Security Manager, perform the following steps.

Step by Step Procedure

In CANoe DE:

  1. Open the Options dialog and navigate to the Bus Systemes/Protocols|Car2x page.

  2. Click [Export...] in the Certificate Manager section. The Certificate Export dialog appears.

  3. Select a path of your choice and click [Export] to export the certificates to this location.

  4. Close the Options dialog and open the Vector Security Manager via the menu path Tools|Security Manager.

In the Vector Security Manager:

  1. In the Security Profile Explorer, select the V2X PKI node.

  2. Click Add (Security Profile) in the Home menu to create a new security profile.

  3. Select the created profile in the Security Profile Explorer and click Add (Parameters) in the Home menu. The Add Parameter dialog appears.

  4. Select the security standard for your region and click [OK] to create a V2X Certificate Group.

  5. Select V2X Certificates in the created V2X Certificate Group and click Add (Parameters) in the Home menu. The Add Parameter dialog appears.

  6. Activate Import File and select the certificates exported in step 3 under Path.

  7. Confirm the dialog with [OK]. The certificates are now displayed in the overview under V2X Certificates.

Switch back to CANoe DE:

  1. Open the Security Configuration via the menu path Simulation|Security Configuration.

  2. Switch to the V2X page and select your security profile. Confirm your selection with [OK].

  3. Open the Options dialog and navigate to the Bus Systemes/Protocols|Car2x page.

  4. Select your certificates in the Certificate Manager section and click [Remove] to delete them.

You have now successfully migrated your configuration to the Vector Security Manager.
