GL Logger Suite

Ribbon: Tools Tab|GL Logger Suite

Valid for:  CANoe DE  •  CANoe:lite DE

General

You can use some loggers as network interface devices in your CANoe DE product. The logger is connected to the CANoe DE product computer via Ethernet connection. Once the measurement begins, the logger sends the data to the CANoe DE product. The data can then be analyzed in the Measurement Setup. You cannot send data.

Monitoring Mode

Signal Sampling Mode

In Monitoring Mode, the logger transfers all incoming data such as messages and Error Frames. During the measurement, the logger does not record any data in the ring buffer. Classifications are also continued during the measurement.

In Signal Sampling Mode, the logger transfers sampled signal values of predefined signals. Each signal is mapped onto a system variable that is created in the GLLogger namespace. During the measurement, the logger continues to record data in the ring buffer.

General

You can configure the data loggers of the GL1000, GL2000, GL3000, GL4000 and GL5000 series with the Vector Logger Suite.
If installed, you can start the Vector Logger Suite via the Tools ribbon tab.
