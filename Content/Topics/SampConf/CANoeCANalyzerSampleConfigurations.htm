Sample Configurations of the CANoe DE product Demo Version

  • The Demo Version is fully functional but without access to any network interface. The time management is done with Windows timers. This means that the simulation time of the demo version does not necessarily correspond to realtime.

Sample Configurations of the CANoe DE product Full Version

Some of the sample configurations can be executed with the 32-bit variant as well as with the 64-bit variant of the realtime kernel (see compatibility overview).

Remind that you need the corresponding CANoe DE product licenses for the bus systems to run the sample configurations for this bus system; e.g. you need a CAN license for the CAN sample configurations.

For some sample configurations, an additional setup is required. So in case of MATLAB Integration Package Setup and LabVIEW Setup (see Additional Components).

General

Our sample configurations are easy to use. They demonstrate many features of your CANoe DE product and contain practical advises for usage.

Load Sample Configurations...
...via CANoe DE product
  1. Start your CANoe DE product.
  2. Open the File ribbon tab and click on (load configuration).
  3. The Open dialog appears.
  4. Open the subfolder you want.
  5. Open the CFG configuration file you want.
  6. The configuration will be opened in your CANoe DE product.

Load Sample Configurations...
...via the folder in the setup directory

Depending on the installation, the folder for sample configurations is stored in different locations. You find it via this link in your file system: Open Folder

With a double click you can open the CFG configuration file you want in your CANoe DE product.
