Operating Concept

This guide is aimed at anyone who would like a quick introduction to the functionality of CANoe DE. It focuses on use with Ethernet networks.

You can open many help pages for dialogs and windows simply by pressing <F1> or the [Help] button. Use the search input field and the search filter at the top right of the help to find the information you need on the functionality of the program. Also read the tips on using the help.

CAPL

CANoe DE offers you a built-in programming language to extend the functionality for user-specific tasks. This language is called CAPL (Communication Access Programming Language). It is similar to C and is also presented here.

  • Extending a Simulation

Vector Hardware

To connect to the network (VN) interfaces, you must install the appropriate Vector hardware drivers.

For this beginner's tour, install the latest driver for the VN interface you are using. You can download this from the Vector Download-Center.
