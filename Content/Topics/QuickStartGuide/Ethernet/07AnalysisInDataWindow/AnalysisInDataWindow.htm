Analyse Signal Values in the Data Window

 

Valid for:  CANoe DE

In addition to the symbolic message names, signal values can also be analyzed with the assigned database. The Data Window (Measurement desktop or Analysis ribbon tab|Data) is used to analyze the current signal values.

This makes it clear why the Data Window is initially empty in a new configuration: The signal values to be displayed depend on the information from the database, among other things. As the user, you must decide which signal values are to be displayed.

Add Signals in the Data Window

To display signal values in the Data Window, you must add signals in the Data Window.

Step by Step Procedure

  1. Open the Symbol Selection dialog via the context menu command Add Signals... of the Data Window.
    The tree view within the dialog allows you to search for a specific signal. Each database has a branch for signals, messages and nodes.
    Multiple selection is possible with <Ctrl>+<Signal>.

  2. Select the signals VehicleSpeed and EMotorCurrent.

  3. Close the dialog with [OK].
    The signal names are now displayed in the Data Window.

You can also drag and drop signals from the Symbol Explorer or other windows (e.g. Trace Window) into the Data Window.

Output in the Data Window

Now start the replay of the logging file either completely or step by step with the set step width of 2 s.
