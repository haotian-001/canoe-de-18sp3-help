Operating Concept

If you start your CANoe DE for the first time, and its functionality and controls are still completely new to you, the following tour will help you to become familiar with its operating concept and its most important features.

For this tour you will first set up a very simple CAN bus where your CANoe DE acts as both, sender and receiver.

  • Create a New Configuration

  • Windows

  • Set Up the Bus

Set Up CANoe DE

In the first step CANoe DE is configured as a data source, i.e. as a transmitting station. You will then learn about analysis options by studying the generated data in the analysis windows afterwards.

In complex real systems CANoe DE typically also acts as sender and receiver. You can utilize the program as a data source to transmit data to other controllers, but you can simultaneously use it to observe, log and evaluate the data traffic on the CAN bus.

  • Transmit Data Without Database

  • Work with Symbolic Data

  • Analyze Signal Values in Data Window

  • Analyze Signal Responses in Graphics Window

  • Log a Measurement

  • Evaluate a Logging File

CAPL

In the last part of the tour you will become familiar with the CAPL programming language and create two network nodes of a distributed system to solve a simple simulation task.

  • Create a CAPL Program

  • Simulate Distributed Systems in CANoe
