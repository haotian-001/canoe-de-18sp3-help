Description of the Windows

 

Valid for:  CANoe DE  •  CANoe:lite DE  •  CANoe4SW DE  •  CANoe4SW:lite DE

The Panel Designer provides the following windows:

Screenshot

Short Description

Working Area

Panels are configured in the working area.

Every time a new panel is opened, a tab appears (this means that multiple panels can be edited in parallel).

Symbol Explorer

All available symbols are shown in the Symbol Explorer.

They can be assigned to an open panel using drag and drop.

Toolbox

All available controls are displayed in the toolbox.

They can be assigned to an open panel using drag and drop or by double-clicking.

Properties

The Properties Window lists all settings associated with the currently selected control.

When a setting is selected, a short description of it appears at the bottom of the Properties Window.
