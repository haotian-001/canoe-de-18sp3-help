Valid for:  CANoe DE  •  CANoe:lite DE  •  CANoe4SW DE  •  CANoe4SW:lite DE

Project Documents

Here you can see:

  • all panels are displayed that are assigned to the CANoe DE product configuration.

    • With the context menu command Remove From Project you can remove panels from the currently loaded CANoe DE product configuration. The panels themselves are not deleted.

  • all FDX files are displayed that are assigned to the CANoe configuration. (available for CANoe)

    • With the context menu command Remove From Project you can remove FDX files from the currently loaded CANoe configuration. The FDX files themselves are not deleted.

  • all vCDL files are displayed that are assigned to the CANoe DE product configuration.

    • With the context menu command Remove From Project you can remove vCDL files from the currently loaded CANoe configuration. The vCDL files themselves are not deleted.

Unassigned Documents

Here, all files are displayed that have been not assigned to CANoe DE product configuration the yet.

With the context menu command Add To Project, you can add

  • a panel to the currently loaded CANoe DE product configuration
  • a FDX file to the currently loaded CANoe configuration. (available forCANoe)

    • You can then see the added FDX descriptions in the options of the running CANoe application under Options.

  • a vCDL file to the currently loaded CANoe DE product configuration.
